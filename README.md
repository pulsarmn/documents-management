# Document Management System

Desktop приложение для создания, загрузки, просмотра и сохранения финансовых документов. Разработано в качестве тестового задания на стажировку.

## Технологический стек

- **Java 25**
- **JavaFx**
- **Jackson**
- **Gradle**

## Ключевые возможности

- **Управление документами:** Создание, просмотр деталей и удаление документов различных типов.
- **Сохранение в формате JSON** Сохранение и загрузка списка документов с сохранением точных типов через служебные метаданные.

## Установка и запуск

**Требования:** Установленная **Java 25**

1. Клонируйте репозиторий

```bash
git clone https://github.com/pulsarmn/documents-management

cd documents-management
```

2. Запустите приложение

```bash
./gradlew run
```
